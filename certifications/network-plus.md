### Overview
CompTIA Network+ validates hands-on skills in designing, configuring, managing, and troubleshooting wired and wireless networks. This certification demonstrates a strong foundation in networking concepts critical to cybersecurity, cloud, and infrastructure roles.

### Skills Demonstrated
- Network architecture (LAN, WAN, WLAN, VLANs)
- IP addressing, subnetting, and routing
- Network protocols and ports (TCP/IP, UDP, DNS, DHCP, HTTP/S)
- Network security fundamentals (firewalls, VPNs, NAC)
- Troubleshooting connectivity and performance issues
- Cloud and virtualization networking concepts
- Network monitoring and documentation

### Tools & Concepts
- OSI & TCP/IP models
- Routers, switches, access points
- Command-line tools (ping, tracert, ipconfig)
- Wireless standards (802.11)
- Network diagrams and topology mapping

### Related Coursework / Experience
- Hands-on troubleshooting simulations (PBQs)
- Network documentation and discovery exercises
- Real-world troubleshooting scenarios involving routing, switching, and wireless performance

### Why This Matters
Network+ provides the technical backbone for my cybersecurity and GRC path, enabling me to assess network risks, understand attack surfaces, and collaborate effectively with infrastructure and security teams.
